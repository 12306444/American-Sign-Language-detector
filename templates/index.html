Project Details

SignAI is an advanced American Sign Language detection system that uses Convolutional Neural Networks (CNN) to recognize ASL gestures in real-time. The model is trained on 87,000 images of ASL alphabet signs, achieving 98% accuracy on the test set. Built with TensorFlow, OpenCV, and MediaPipe, this system aims to bridge communication gaps for the deaf and hard-of-hearing community.
